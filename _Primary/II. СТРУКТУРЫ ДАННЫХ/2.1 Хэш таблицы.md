### **Хэш таблица** - структура данных, связывающая значения по ключу (хэшу)

Должна выполнять 3 операции:
1. добавление пары
2. удаление пары
3. поиск по ключу

**Хэш** - битовая строка определенной длины, созданная алгоритмом на основе входного массива данных.
Алгоритмы хэширования:

**MD5**: Message Digest 5, 128 бит
**RIPEMD-160**: 160 бит
**Whirlpool**: 256 бит
**SHA**: Secure Hash Algorithms, сжатие
	SHA-0, SHA-1 - 160 бит
	SHA-2 - _SHA-256_, _SHA-512_ (Davies–Meyer structure, Merkle–Damgård structure разбиение входных сообщений произвольной длины на блоки фиксированной длины и работа с ними по очереди с помощью функции сжатия, каждый раз принимая входной блок с выходным от предыдущего прохода.)
	SHA-3 - переменная длина входа, Keccak sponge - перестановки
